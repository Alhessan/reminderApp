# Reminder App (Ionic)

Ionic + Angular application with Capacitor for native builds.

## Prerequisites

- Node.js (v18+)
- npm or yarn
- For Android: Android Studio, JDK 17+
- For iOS: Xcode (macOS only), CocoaPods

## Run commands

### Web (browser)

```bash
npm install
npm start
```

Open [http://localhost:4200](http://localhost:4200). Uses in-memory SQLite-style store with localStorage persistence.

### Build for production

```bash
npm run build
```

Output is in `www/` (used by Capacitor).

### Native (Android / iOS)

Sync the web build and run on a device or emulator:

```bash
npm run build
npx cap sync
```

**Android**

```bash
npx cap open android
```

Then run the app from Android Studio, or:

```bash
npx cap run android
```

**iOS** (macOS only)

```bash
npx cap open ios
```

Then run from Xcode, or:

```bash
npx cap run ios
```

### Other scripts

| Command        | Description                    |
|----------------|--------------------------------|
| `npm run watch`| Build and watch for changes    |
| `npm test`     | Run unit tests (Karma)         |
| `npm run lint`  | Run ESLint                     |

## Troubleshooting

If the **task detail page** stays on "Loading…" even when the console shows load complete, see [docs/DEBUGGING-TASK-DETAIL.md](docs/DEBUGGING-TASK-DETAIL.md) for a short step-by-step to find the cause (canary block + binary-search in the template).

## Project structure

- `src/app/` — Angular app (pages, services, models)
- `android/` — Capacitor Android project
- `ios/` — Capacitor iOS project (after `cap add ios`)
- `www/` — Built web assets (generated by `npm run build`)
